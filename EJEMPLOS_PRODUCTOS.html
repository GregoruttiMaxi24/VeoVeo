Ejemplos de Cómo Agregar Productos

Ejemplo 1: Producto Simple

Para agregar un nuevo producto, usa esta estructura:

<div class="col-12 col-md-6 col-lg-3">
    <div class="guitarraVenta d-flex flex-column">
        <h3 class="fw-bold">Nombre del Producto</h3>
        <img class="img-fluid" src="img/producto.jpg" alt="Nombre">
        <p class="text-light lh-sm">Descripción del producto...</p>
        <h4 class="mt-auto fw-bold text-light">Precio: $<span class="precio">99.99</span></h4>
        <button class="btn btn-lg mt-auto btn-primary comprar-btn" 
                data-id="6" 
                data-nombre="Nombre del Producto" 
                data-precio="99.99">
            Comprar
        </button>
    </div>
</div>
Atributos Importantes:
  • data-id: ID único del producto (debe ser diferente a los otros)
  • data-nombre: Nombre exacto del producto
  • data-precio: Precio numérico (ej: 99.99)

Ejemplo 3: Agregar Imagen del Producto

Paso 1: Guardar imagen

Coloca la imagen en la carpeta img/

Formatos recomendados: JPG, PNG

Tamaño recomendado: 300x400px

Paso 2: Usar en HTML
<img class="img-fluid" src="img/vestido-rojo.jpg" alt="Vestido Rojo">
Atributos:
  • src: Ruta de la imagen
  • alt: Texto alternativo (para accesibilidad)
  • class="img-fluid": Hace que se adapte al contenedor

Ejemplo 4: Cambiar Precios

En el HTML:
<h4 class="mt-auto fw-bold text-light">
    Precio: $<span class="precio">50.00</span>
</h4>
Y en el botón:
<button ... data-precio="50.00" ...>Comprar</button>

Importante: El precio en el span y en data-precio debe ser exacto

Ejemplo 5: Agregar Ofertas/Descuentos

Mostrar Precio Original y Descuento:
<h4 class="mt-auto fw-bold text-light">
    Precio Original: <del>$100.00</del>
    <br>
    Ahora: $<span class="precio text-danger">79.99</span>
    <span class="badge bg-danger">-20%</span>
</h4>

Recuerda: El precio en data-precio debe ser el precio final (con descuento)

Ejemplo 6: Agregar Especificaciones

Mostrar especificaciones del producto:

    </p>

        Precio: $50.00
    </h4>
    
    <button class="btn btn-lg mt-auto btn-primary comprar-btn" ...>
        Comprar
    </button>
</div>

Ejemplo 7: Gestionar IDs de Productos

Cada producto necesita un ID único:

Producto ID Estado
Vestido 1 1 ✅ Usado
Vestido 2 2 ✅ Usado
Vestido 3 3 ✅ Usado
Vestido 4 4 ✅ Usado
Vestido 5 5 ✅ Usado
Mi Nuevo Producto 6 ❌ Disponible
Otro Producto 7 ❌ Disponible

Regla: Siempre usa el siguiente número disponible

Ejemplo Completo: Copiar y Pegar

Copia este código completo en index.html después de Vestido 5:

<div class="col-12 col-md-6 col-lg-3">
    <div class="guitarraVenta d-flex flex-column">
        <h3 class="fw-bold">Vestido Clásico</h3>
        <img class="img-fluid" src="img/vestido-clasico.jpg" alt="Vestido Clásico">
        <p class="text-light lh-sm">
            Un elegante vestido clásico perfecto para cualquier ocasión. 
            Cómodo, elegante y versátil.
        </p>
        <h4 class="mt-auto fw-bold text-light">
            Precio: $<span class="precio">85.00</span>
        </h4>
        <button class="btn btn-lg mt-auto btn-primary comprar-btn" 
                data-id="6" 
                data-nombre="Vestido Clásico" 
                data-precio="85.00">
            Comprar
        </button>
    </div>
</div>

Paso 1: Cambia "6" por el siguiente ID disponible
Paso 2: Cambias el nombre del producto
Paso 3: Cambias el precio
Paso 4: Cambias la ruta de la imagen
Paso 5: ¡Guarda y recarga!
